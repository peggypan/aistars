# 🌟 AI Stars - 产品需求文档 (PRD) V2

> 基于竞品调研后的产品定位调整

---

## 🎯 产品定位调整

### 原定位
"AI虚拟明星管理平台" → 过于宽泛，与虚拟偶像平台直接竞争

### 新定位
**"AI影视演员库 - 为独立创作者提供数字演员资源"**

```
🎬 核心场景：独立 filmmaker 打开 AI Stars，像选角一样挑选虚拟演员
```

### 一句话价值主张
> "零成本拥有专属演员阵容，让一个人的电影工作室成为可能"

## 📊 竞品差异化策略

### 竞争格局

| 象限 | 高工具属性 | 低工具属性 |
|------|-----------|-----------|
| **高IP运营** | A-SOUL/柳夜熙<br>(大厂资源，难竞争) | 洛天依<br>(成熟IP，无法复制) |
| **低IP运营** | **← AI Stars 定位**<br>(工具+演员库) | Character.AI<br>(纯聊天，无作品) |

### 核心差异化

| 维度 | 传统虚拟偶像 | 数字人工具 | **AI Stars** |
|------|-------------|-----------|-------------|
| 角色来源 | 公司设计 | 真人克隆 | **AI生成原创** |
| 角色深度 | 形象+人设 | 形象+声音 | **+完整作品履历** |
| 使用场景 | 粉丝运营 | 播报/客服 | **影视制作** |
| 与制作工具 | ❌ 无 | ❌ 无 | **✅ FilmStudio集成** |
| 成本 | 百万级制作 | 千元级克隆 | **零成本生成** |

## 👤 目标用户画像

### 主要用户：独立创作者

```
姓名：小林
年龄：28岁
职业：独立 filmmaker / 短视频创作者
痛点：
- 请不起演员，拍摄成本高
- 找不到符合角色气质的演员
- 演员时间难协调，拍摄进度慢
- 想拍科幻/奇幻题材，但找不到合适外貌的演员

使用场景：
"想拍一个赛博朋克风格的短片，需要几个有未来感的角色。
打开 AI Stars，选择'科幻演员'分类，生成了3个不同风格的AI演员。
直接导入 FilmStudio 开始制作。"
```

```
团队：3人短视频工作室
需求：需要固定出镜的"主持人"，但不想绑定真人
方案：在 AI Stars 创建一个虚拟主持人IP
优势：永不塌房、随时可用、形象可控
```

### 探索用户：AI艺术家

```
需求：探索AI角色的艺术可能性
使用：生成独特风格的虚拟角色，用于AI艺术创作
```

### MVP 核心功能 (4周)

#### 1. 演员档案管理
- 创建/编辑/删除演员
- 基本信息：姓名、年龄、性别、国籍
- 人物设定：性格标签、背景故事、特长技能
- 形象描述：外貌、风格、标志性特征
- 分类标签：武打明星/歌手/主持人等

#### 2. AI生成演员
- 提示词输入生成完整档案
- 内置提示词模板（科幻风、古风、现代等）
- 生成参数：年龄范围、性别、风格
- 支持重新生成/迭代优化

#### 3. 基础演员库
- 网格/列表双视图
- 分类筛选 + 搜索
- 演员详情页
- 收藏功能

#### 4. FilmStudio 导出
- 导出演员为 FilmStudio Cast 格式
- 基础字段映射
- 一键导入按钮

## 🎨 用户体验流程

### 核心用户旅程

```
[发现需求] → [寻找演员] → [生成/选择] → [查看详情] → [导出使用]
     │           │            │            │           │
     ▼           ▼            ▼            ▼           ▼
 想拍短片    打开演员库    AI生成或      确认人设     导入
 缺演员      筛选分类      选择现有      查看作品     FilmStudio
```

### 详细流程设计

#### 场景1：AI生成新演员

```
1. 用户点击"生成演员"
2. 选择模板或输入提示词：
   "赛博朋克风格的女主角，25岁，黑客，性格冷酷但内心善良"
3. AI生成：
   - 姓名：艾琳 (Irene)
   - 基本信息：25岁，女性，国籍未知
   - 性格：冷静、理性、独立、内心柔软
   - 背景：地下黑客组织成员...
   - 特长：编程、电子战、格斗
   - 外貌：银白色短发，机械义眼...
4. 用户可编辑/调整
5. 保存到演员库
6. 可选：生成头像图片
```

#### 场景2：从演员库选角

```
1. 用户进入演员库
2. 筛选：类型=科幻演员 + 性别=女性 + 年龄=20-30
3. 浏览结果，查看详情
4. 对比多个候选演员
5. 选择"艾琳"
6. 点击"导出到 FilmStudio"
7. 在 FilmStudio 中 casting
```

#### 场景3：管理演员作品

```
1. 在 FilmStudio 完成一部短片
2. 标记参演演员
3. 数据同步回 AI Stars
4. 艾琳的档案中增加作品：
   - 《霓虹之夜》(2026)
   - 角色：女主角/黑客
5. 作品履历丰富，提升演员价值
```

## 📋 数据模型调整

### 简化版 MVP 模型

```typescript
// Star - 演员 (简化版)
interface Star {
  id: string
  name: string              // 艺名
  avatar?: string           // 头像URL (可选，MVP可先用占位图)
  
  // 基础信息
  age: number
  gender: 'male' | 'female' | 'other'
  nationality: string
  
  // 人物设定 (核心)
  personality: string[]     // 性格标签 ["冷静", "理性"]
  background: string        // 背景故事
  skills: string[]          // 特长 ["编程", "格斗"]
  
  // 形象 (文本描述)
  appearance: string        // 外貌描述
  style: string            // 穿搭风格
  signature?: string        // 标志性特征
  
  // 分类
  categories: string[]      // 分类ID数组
  
  // 作品 (简化)
  movies: {
    id: string
    title: string
    roleName: string
    year: number
  }[]
  
  // AI生成信息
  aiGenerated: boolean
  aiPrompt: string          // 生成提示词 (保留用于迭代)
  
  // 统计
  viewCount: number
  createdAt: Date
  updatedAt: Date
}

// Category - 分类
interface Category {
  id: string
  name: string              // 武打明星、科幻演员等
  type: 'actor' | 'singer' | 'host' | 'other'
  icon?: string
  description?: string
  starCount: number
}

// Movie - 作品 (简化，MVP主要手工录入)
interface Movie {
  id: string
  title: string
  year: number
  type: 'movie' | 'short' | 'ad'
  poster?: string
  stars: string[]           // 参演演员ID
}
```

## 🎬 与 FilmStudio 集成方案

### 数据流向

```
AI Stars                    FilmStudio
   │                            │
   │  1. 导出演员信息            │
   │ ─────────────────────────> │
   │  {                         │
   │    name,                   │
   │    avatar,                 │
   │    bio,                    │
   │    aiStarId                │
   │  }                         │
   │                            │
   │  2. 作品完成同步            │
   │ <───────────────────────── │
   │  {                         │
   │    movieTitle,             │
   │    starId,                 │
   │    roleName                │
   │  }                         │
   │                            │
```

### API 设计

```typescript
// 导出演员 (FilmStudio 调用)
GET /api/stars/:id/export
Response: {
  id: string
  name: string
  avatar?: string
  type: 'actor'
  bio: string           // 整合 background + skills
  skills: string[]
  aiStarId: string      // 关联ID
  metadata: {
    personality: string[]
    appearance: string
    age: number
    gender: string
  }
}

// 批量导出
POST /api/stars/export-batch
Body: { ids: string[] }

// 同步作品 (FilmStudio 回调)
POST /api/stars/:id/movies/sync
Body: {
  movieId: string
  title: string
  year: number
  roleName: string
  roleType: string
}
```
